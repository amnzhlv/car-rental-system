# Car Rental System

A full-stack web application for managing car rentals with user and admin access.

## Tech Stack
- **Frontend:** React, jQuery, Web Storage, JSON
- **Backend:** Node.js + Express
- **Database:** MySQL
- **Extras:** JSON Web Token authentication, Axios for API calls, Tailwind CSS styling
